# Airbrush
A gesture-controlled drawing application that allows users to draw in the air using their fingers instead of a physical brush or mouse.

This project was created as a learning project to explore computer vision, hand tracking, and real-time interaction using Python.

The system tracks hand movements through a webcam and converts finger motion into digital drawing on screen.

# Note

This project was built for educational and practice purposes while learning computer vision and interactive Python applications.

# *Features:*

Finger-tracking based drawing

Webcam-powered interaction

Real-time motion detection

Virtual canvas rendering

Touch-free drawing experience

Basic gesture recognition

# *Tech Stack*

Python

OpenCV

MediaPipe
